Tags for HTML

Listed below are the different tags that are used for HTML...

The following tags are all within < and >:

!DOCTYPE html - Defines document type as html

head - Everything that is unseen but goes on top of browser

title - The title that will be seen on the top of browser

body - Everything that is seen on the website

h(number) - Heading

p - Paragraph

br/ - Line break

hr - Horizontal rule

a href="#(name)" - Link within page

a title="(tooltip)" - Provides tool tip for link

a href="(url)" - Link to url or any webpage within same folder

a name="(name)" - Sets target destination for hyperlink

!--"(comment)"-- - Comment

img src="(name of image file)" height="(number of pixels)" width= "(number of pixels)"/ - Image from source

table border="number of pixels" - Make a table with border attributes

tr - Table Row

td - Table Data (Columns)

th colspan="(Number of Pixels)" - Table Header (works like table data) where col span shows how many columns data spans

th cellpadding="(Number of Pixels)" - Cell Padding gives space within cell

th cellspacing="(Number of Pixels)" - Cell Spacing gives space between cells

ul - Unordered List

ol - Ordered List

li - List item

style type="text/css"- CSS adding style to text
